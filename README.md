# lab01-AFD
Este programa deve receber como entrada um autômato finito determinístico M = (Σ, Q, q0, δ, F), analisar cada palavra da entrada e produzir como saída a mensagem “M aceita a palavra &lt;palavra>” se a &lt;palavra> for aceita pelo autômato, ou a mensagem “M rejeita a palavra &lt;palavra>”, caso a &lt;palavra> for rejeitada.
